# Portofolio.clone.elektric
Proyek ini merupakan clone website Electromaker, dibuat untuk menampilkan produk dan artikel teknologi dengan tampilan modern dan responsif. Website dikembangkan menggunakan HTML, CSS, dan Bootstrap untuk meniru struktur, warna, serta tata letak aslinya. Tujuan proyek ini adalah melatih kemampuan desain antarmuka, penataan layout, dan penggunaan
